# Tech Stack

## Constraints
- **NO FRAMEWORKS:** Vanilla JS, HTML, CSS only. (Reduces build complexity, faster on low-end hardware).
- **Backend:** Node.js (Express).
- **Persistence:** Supabase (Primary) + Local JSON (Backup).

## Dependencies (Locked)
- **Node.js:** v18+
- **Express:** `^4.18.2`
- **Supabase JS:** `^2.39.0`
- **Cors:** `^2.8.5`
- **Dotenv:** `^16.3.1`

## Environment
- **Server:** macOS (Mac Mini).
- **Client:** Chrome/Safari on iPad/Android Tablets.
- **Port:** 8083 (Default).

## External Services
- **Supabase:** Database, Auth, Realtime.
- **Cloudflare:** Tunnel (Remote Access).
